# Tutorials

These tutorials are a hands-on introduction to CEDAR, taking you from building
your first template to authoring metadata with an AI assistant. Each one is
self-contained and works through a complete example. Read them in order or jump
straight to the one you need.

- [**CEDAR Tutorial**](cedar_tutorial.md): build a template and fill it out by
  hand in the CEDAR Workbench, then publish a versioned release and share an
  instance on the public web. Start here if you are new to CEDAR.
- [**CEDAR Controlled Term Tutorial**](cedar_term_tutorial.md): constrain template
  fields to ontology terms from BioPortal, binding a field to a whole ontology, a
  branch, or a set of individual classes you assemble yourself.
- [**CEDAR MCPs Tutorial**](cedar_mcps_tutorial.md): build the same kind of
  template by conversation with an AI assistant, using the CEDAR and BioPortal MCP
  servers to resolve terms, assemble the template, and preview it.
