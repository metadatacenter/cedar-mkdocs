# Introduction

The CEDAR User Guide is a practical introduction to working with the CEDAR Workbench, the platform for authoring, standardizing, and sharing scientific metadata. It is written for everyday users — researchers, data curators, and template designers — and assumes no prior experience with CEDAR.

The guide starts with the basic tasks: creating an account and signing in, searching for and organizing resources, viewing the contents and metadata of those resources, moving around your workspace, filling out metadata forms, and sharing your work with collaborators or the public.

It then turns to CEDAR templates, the reusable forms that give metadata its structure. These chapters explain what a template is and how it is organized, describe building one field by field, and describe adding controlled terms and ontology-based semantics in line with FAIR principles. Further chapters cover advanced template techniques and tips for designing templates that others can reuse.

The remaining parts address more advanced topics for all users, including how CEDAR identifies resources with persistent IRIs and how to interact with the system programmatically through its REST API. A dedicated section supports caDSR users working with Common Data Elements, and reference material — an orientation to the Workbench, frequently asked questions, and a glossary — rounds out the manual.
