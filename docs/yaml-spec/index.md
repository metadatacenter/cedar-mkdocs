# CEDAR YAML Specification

This specification defines the YAML serialization of the CEDAR model, the standards-based
model with which scientific communities represent their metadata requirements precisely and
in a form software can act on. It is the reference for writing and reading CEDAR metadata as
YAML.

The model is defined first, on its own terms and independently of any serialization. The
rest of the specification defines the YAML for this model, construct by construct. It closes
with complete worked artifact examples.
