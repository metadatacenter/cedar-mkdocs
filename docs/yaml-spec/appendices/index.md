# Appendices

Reference material supporting the specification, gathered for quick lookup.

- **[Key Index](key-index.md)** lists every key alphabetically, with the page that defines
  each.
- **[Controlled Vocabularies](controlled-vocabularies.md)** collects the fixed sets of tokens
  a key may take, such as the artifact types, statuses, and datatypes.
- **[Datatypes](datatypes.md)** describes the XSD datatypes used to type numeric and temporal
  values.
- **[Glossary](glossary.md)** defines the core terms used throughout.
