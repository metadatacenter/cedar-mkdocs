# Overview

The CEDAR Developer Guide is for developers who want to work with CEDAR programmatically rather than through the web interface.

It builds on the [CEDAR model](../yaml-spec/cedar-model.md), which defines the representation of all CEDAR metadata artifacts — templates, elements, fields, and the metadata instances themselves.

Building on that model, the guide gets you started with the CEDAR REST APIs for creating, validating, and retrieving these artifacts and interacting with CEDAR services directly.
