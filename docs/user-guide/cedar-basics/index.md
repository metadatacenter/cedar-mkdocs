# CEDAR Basics

These chapters describe the basic tasks involved in using CEDAR.

They begin with creating an account and signing in, followed by an overview of the CEDAR workspace — the view presented after logging in.

The next chapters cover finding resources by searching and narrowing the results, and viewing a resource's contents and metadata both within CEDAR and on the web. The final chapters describe managing your content — copying, moving, renaming, and deleting resources — and sharing it with groups or publishing it for others to read or reuse.
