# CEDAR Basics

These chapters describe the everyday tasks involved in using CEDAR.

They begin with creating an account and signing in, followed by an overview of the CEDAR workspace — the view presented after logging in. The next chapters turn to templates, the reusable forms that structure CEDAR metadata: what a template is, how to fill one out to create and validate metadata, how to build a basic template field by field, and how to add controlled terms and ontology-based semantics in line with FAIR principles.

The remaining chapters cover finding resources by searching and narrowing the results, managing your content — copying, moving, renaming, and deleting resources — and sharing it with groups or publishing it for others to read or reuse.
