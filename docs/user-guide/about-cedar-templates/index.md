# About CEDAR Templates

Templates are the reusable forms that define the structure of CEDAR metadata. These chapters describe how to design them.

They begin with what a template is and how it is organized, then describe building a basic template field by field, and adding controlled terms and ontology-based semantics in line with FAIR principles.

The later chapters cover advanced template topics — extending templates, versioning, working with common data elements, and the suggestion system.
