# Advanced Topics

These chapters cover topics beyond the basic use of the workbench.

They include viewing a resource's contents and metadata — within CEDAR, on the web, and as raw JSON; how CEDAR identifies each resource with a persistent IRI, and how to construct an IRI that opens a metadata form for a given template; interacting with the system programmatically through CEDAR's REST API; working with Common Data Elements (CDEs) — finding, browsing, and maintaining the standardized fields you can reuse across templates; and guidance aimed specifically at caDSR users.
