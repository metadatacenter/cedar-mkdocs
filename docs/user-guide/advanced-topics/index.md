# Advanced Topics

These chapters cover topics beyond the basic use of the workbench.

One covers working with Common Data Elements (CDEs) — finding, browsing, and maintaining the standardized fields you can reuse across templates. Another describes how CEDAR identifies each resource with a persistent IRI, how to read those identifiers, and how to construct an IRI that opens a metadata form for a given template. A third introduces CEDAR's REST API for interacting with the system programmatically to find and retrieve metadata.
